# CGAL Jupyter Lab

A ready-to-go Jupyter Lab Backend with CGAL environment

## Launch Jupyter Lab Backend
```sh
git clone https://github.com/camilleperin/cgal-jupyter-lab.git
cd cgal-jupyter-lab
docker-compose up
```
